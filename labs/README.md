# Лабораторные данные (офлайн-практика)

Файлы в этой папке позволяют выполнять задания **без** реальных сетевых устройств.

| Файл | Назначение |
|------|------------|
| [inventory.json](inventory.json) | Пример инвентаря для JSON/YAML/Nornir |
| [inventory.yaml](inventory.yaml) | То же в YAML |
| [nornir_config.yaml](nornir_config.yaml) | Пример конфигурации Nornir (`SimpleInventory`) |
| [nornir/hosts.yaml](nornir/hosts.yaml) | Хосты для Nornir (пароль задайте в коде через `defaults` или переменные окружения) |
| [sample_outputs/](sample_outputs/) | Текстовые «выводы» CLI, XML, пример JSON фактов |

Пути в заданиях указаны относительно корня курса `pythonAI/`.
