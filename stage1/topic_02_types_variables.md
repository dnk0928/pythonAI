## Теория

### Переменные — это ссылки

В Python переменная — это **имя**, которое ссылается на объект в памяти. Тип — у объекта, не у имени.

```python
x = 10          # x ссылается на объект int(10)
x = "hello"     # теперь x ссылается на объект str("hello")
y = x           # y и x ссылаются на один и тот же объект
```

Функция `type()` возвращает тип объекта, `isinstance()` проверяет принадлежность:

```python
type(42)          # <class 'int'>
type("hello")     # <class 'str'>
isinstance(42, int)    # True
isinstance(42, (int, float))  # True — проверка на несколько типов
```

### Числа: int и float

```python
# int — целые числа произвольной точности
port = 22
mask = 0xFF          # шестнадцатеричный литерал = 255
vlan = 0b1010        # двоичный = 10
octets = 1_000_000   # подчёркивания для читаемости

# float — числа с плавающей точкой (IEEE 754)
latency = 1.5
rtt = 3.14e-3        # научная нотация = 0.00314

# Деление
10 / 3     # 3.3333... (всегда float)
10 // 3    # 3 (целочисленное деление)
10 % 3     # 1 (остаток)
2 ** 10    # 1024 (возведение в степень)
```

### bool — логический тип

`bool` — подкласс `int`. `True == 1`, `False == 0`.

```python
True + True   # 2 (не делайте так в реальном коде, но знать полезно)
bool(0)       # False
bool(1)       # True
bool("")      # False — пустая строка
bool("0")     # True — непустая строка (не путайте с int(0)!)
bool([])      # False — пустой список
bool([0])     # True — список с элементом
```

**Falsy-значения** (приводятся к `False`): `None`, `0`, `0.0`, `""`, `[]`, `{}`, `()`, `set()`.

### NoneType

`None` — единственный объект типа `NoneType`. Обозначает «ничего» / «нет значения».

```python
result = None
if result is None:    # всегда используйте "is None", не "== None"
    print("no result")
```

### Операторы

```python
# Арифметические
+  -  *  /  //  %  **

# Сравнения — возвращают bool
==  !=  <  >  <=  >=

# Логические
and  or  not

# Принадлежность
"Gi" in "Gi0/1"      # True
10 not in [1, 2, 3]  # True

# Тождественность (один ли объект в памяти)
x is None
x is not None

# Битовые (важны для масок!)
0b11111111 & 0b11000000   # AND  = 192
0b11000000 | 0b00001111   # OR   = 207
0b11111111 ^ 0b00001111   # XOR  = 240
~0b00001111               # NOT  = -16 (в дополнении до двух)
1 << 3                    # сдвиг влево = 8
256 >> 4                  # сдвиг вправо = 16
```

### Преобразование типов

```python
int("255")          # 255
int("ff", 16)       # 255 — из шестнадцатеричной строки
int("11111111", 2)  # 255 — из двоичной строки
float("3.14")       # 3.14
str(255)            # "255"
bool(0)             # False
hex(255)            # "0xff"
bin(255)            # "0b11111111"
oct(255)            # "0o377"

# Опасность: input() всегда возвращает str
port = input("Port: ")   # "22" — строка, не число!
port = int(input("Port: "))  # правильно
```

### Составное присваивание

```python
x = 10
x += 5    # x = x + 5 = 15
x -= 3    # x = 12
x *= 2    # x = 24
x //= 5   # x = 4
```

### Множественное присваивание и распаковка

```python
a, b = 1, 2          # a=1, b=2
a, b = b, a          # обмен значений без temp-переменной

ip, port = "10.0.0.1", 22    # распаковка кортежа
first, *rest = [1, 2, 3, 4]  # first=1, rest=[2,3,4]

# Walrus-оператор := (Python 3.8+) — присваивание внутри выражения
import re
if m := re.search(r"\d+", line):
    print(m.group())   # используем m сразу
```

## Примеры (сетевые)

```python
# Проверка принадлежности октета диапазону
octet = int("192")
is_valid = 0 <= octet <= 255
print(f"Octet {octet} valid: {is_valid}")

# Маскирование адреса
ip_int = 0xC0A80101       # 192.168.1.1
mask_int = 0xFFFFFF00     # /24
network = ip_int & mask_int
print(hex(network))       # 0xc0a80100 = 192.168.1.0

# Преобразование октета двоичного представления
binary_octet = "11000000"
decimal = int(binary_octet, 2)
print(decimal)            # 192
```

## Частые ошибки

```python
# Сравнение float — НЕ делайте так
0.1 + 0.2 == 0.3   # False! (проблема представления float)
abs(0.1 + 0.2 - 0.3) < 1e-9  # правильно

# None — не то же самое, что пустая строка
x = None
if x == None:  # работает, но не Pythonic
if x is None:  # правильно

# Тип input() всегда str
port = input("port: ")
port + 1  # TypeError! нужен int(port)
```
